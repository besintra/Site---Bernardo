Matemática

Conceitos Matemáticos Fundamentais

Abaixo estão alguns dos conceitos matemáticos fundamentais que aplico em meus projetos de análise de dados e IoT.

Fatorial

O fatorial de um número natural n, representado por n!, é o produto de todos os inteiros positivos menores ou iguais a n.

Fórmula:

n! = n × (n-1) × (n-2) × ... × 3 × 2 × 1

Por definição: 0! = 1

Exemplo:

5! = 5 × 4 × 3 × 2 × 1 = 120

Permutação

Número de maneiras de organizar n elementos distintos em sequência.

Fórmula:

P(n) = n!

Exemplo:

De quantas maneiras podemos organizar 3 livros em uma prateleira?

P(3) = 3! = 6 maneiras

Arranjo

Número de maneiras de selecionar e organizar r elementos de um conjunto de n elementos distintos, onde a ordem importa.

Fórmula:

A(n, r) = n! / (n - r)!

Exemplo:

Quantos códigos de 2 dígitos podemos formar com os números 1, 2, 3, 4?

A(4, 2) = 4! / (4-2)! = 24 / 2 = 12 códigos

Combinação

Número de maneiras de selecionar r elementos de um conjunto de n elementos distintos, onde a ordem não importa.

Fórmula:

C(n, r) = n! / [r! × (n - r)!]

Exemplo:

De quantas maneiras podemos escolher 2 pessoas de um grupo de 5?

C(5, 2) = 5! / [2! × (5-2)!] = 120 / (2 × 6) = 10 maneiras

Probabilidade

Medida da chance de um evento ocorrer, variando de 0 (impossível) a 1 (certo).

Fórmula:

P(A) = Nº de casos favoráveis / Nº total de casos possíveis

Exemplo:

Qual a probabilidade de tirar um ás em um baralho de 52 cartas?

P(ás) = 4 / 52 = 1/13 ≈ 0.077

Juros Simples

Juros calculados apenas sobre o valor inicial (capital).

Fórmula:

J = C × i × t
M = C + J = C × (1 + i × t)

Onde: J = juros, C = capital, i = taxa, t = tempo

Exemplo:

Um empréstimo de R$ 1.000,00 a 5% ao mês por 3 meses:

J = 1000 × 0,05 × 3 = R$ 150,00

M = 1000 + 150 = R$ 1.150,00

Juros Compostos

Juros calculados sobre o capital inicial e também sobre os juros acumulados.

Fórmula:

M = C × (1 + i)t

Onde: M = montante, C = capital, i = taxa, t = tempo

Exemplo:

Um investimento de R$ 1.000,00 a 5% ao mês por 3 meses:

M = 1000 × (1 + 0,05)3 = 1000 × 1,157625 = R$ 1.157,63

Variância

Medidas de dispersão que indicam o quanto os valores de um conjunto de dados se afastam da média.

Fórmulas:

Variância (σ²) = Σ(xᵢ - μ)² / N

Onde: xᵢ = cada valor, μ = média, N = número de elementos

Exemplo:

Para o conjunto {2, 4, 6, 8, 10}:

Média = 6, Variância = 8

Desvio Padrão

Medidas de dispersão que indicam o quanto os valores de um conjunto de dados se afastam da média.

Fórmulas:

Desvio Padrão (σ) = √Σ(xᵢ - μ)² / N

Onde: xᵢ = cada valor, μ = média, N = número de elementos

Exemplo:

Para o conjunto {2, 4, 6, 8, 10}:

Média = 6, Desvio Padrão = √8 ≈ 2,83

Área e Volume de Sólidos Geométricos

Cubo

Área: A = 6 × a²

Volume: V = a³

Onde: a = aresta

Paralelepípedo

Área: A = 2(ab + ac + bc)

Volume: V = a × b × c

Onde: a, b, c = dimensões

Cilindro

Área lateral: Al = 2πrh

Área total: At = 2πr(h + r)

Volume: V = πr²h

Onde: r = raio, h = altura

Esfera

Área: A = 4πr²

Volume: V = (4/3)πr³

Onde: r = raio

Cone

Área lateral: Al = πrg

Área total: At = πr(g + r)

Volume: V = (1/3)πr²h

Onde: r = raio, h = altura, g = geratriz

Pirâmide

Área lateral: Soma das áreas das faces laterais

Área total: Al + Ab

Volume: V = (1/3) × Ab × h

Onde: Ab = área da base, h = altura
